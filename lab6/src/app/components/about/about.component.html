Project Overview

This is a full-featured Angular application built as part of a web development course. It functions as an anime archive with browsable series, photo galleries, and inline editing — built with Angular 17+ standalone components.

Capabilities

  • 100 Anime series catalogued and browsable by regiment
  • Per-series photo galleries with pagination
  • Inline series title editing and commit functionality
  • Entry purge (delete) with local state update
  • Angular Router with standalone component architecture
  • Page memory — returns to last viewed registry page
